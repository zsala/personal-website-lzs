# Personal Website

A modern, static website showcasing professional experience, skills, and portfolio.

## Overview

This repository contains the source code for a personal website built with HTML, CSS, and JavaScript. The site features a clean, responsive design and includes sections for professional experience, education, skills, and project showcases.

## Structure

```
├── assets/          # Images and media files
├── css/             # Stylesheets
├── game/            # Game development showcase
├── privacy/         # Privacy policy pages
├── index.html       # Main homepage
└── stories.html     # Stories section
```

## Features

- Responsive design
- Professional portfolio presentation
- Experience and education sections
- Skills showcase
- Game development portfolio
- Privacy policy pages

## Technologies

- HTML5
- CSS3
- JavaScript
