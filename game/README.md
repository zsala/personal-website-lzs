# The Grand Mouse Escape - Official Website

This is the static website for The Grand Mouse Escape game.

## Structure

```
website/
├── index.html          # Main homepage
├── privacy.html        # Privacy policy page
├── terms.html          # Terms of service page
├── css/
│   └── styles.css      # Main stylesheet with game-themed design
└── assets/             # Game images and icons
    ├── splash-design.png
    ├── phone-1.png
    ├── phone-2.png
    ├── tablet-7-1.png
    ├── tablet-7-2.png
    ├── icon.png
    ├── mouse.png
    ├── cats.png
    ├── cheese.png
    └── house-icon.png
```

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Game-themed Styling**: Uses the same color palette as the privacy policy site
- **Smooth Animations**: Subtle scroll animations and hover effects
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Accessibility**: Good contrast ratios and semantic structure

## Design System

Based on the existing privacy policy site design:

- **Background**: `#1b1412` (dark brown)
- **Primary Text**: `#f4d9a3` (warm cream)
- **Accent Color**: `#ffa644` (orange)
- **Secondary Accent**: `#ffcb7a` (light orange)
- **Cards/Sections**: `#36261f` (medium brown)
- **Font**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif

## Pages Included

1. **Homepage** - Hero section, game features, screenshots gallery
2. **Privacy Policy** - Data collection and privacy information
3. **Terms of Service** - Legal terms and conditions
